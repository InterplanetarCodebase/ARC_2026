# Rocket M2 Configuration

This document contains the current radio and network settings for the ARC 2026 base station and rover links.

## Base Station

### Hardware

| Item | Value |
|---|---|
| Module | Ubiquiti Rocket M2 (2.4 GHz) |
| Antenna | AM-2G15-120 (15 dBi) |

### Login

| Field | Value |
|---|---|
| Username | ubnt |
| Password | interplanetar |

### Wireless Settings

| Setting | Value |
|---|---|
| Wireless Mode | Access Point |
| WDS | Enabled |
| SSID | interplanetar |
| Country Code | United States |
| IEEE Mode | BGN Mixed |
| Channel Width | 40 MHz fixed |
| Frequency | 2442 MHz (based on survey) |
| Extension Channel | Upper |
| Output Power | 27 dBm |
| Max TX Rate | MCS 15 |

### Security

| Setting | Value |
|---|---|
| WPA2-PSK | interplanetar |

### Network Settings

| Setting | Value |
|---|---|
| Gateway | 192.168.1.1 |
| IP Address (Rocket) | 192.168.1.20 |
